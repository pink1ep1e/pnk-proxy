# 💜 PNK Telegram Proxy

**Fast. Secure. Always Online.**

MTProto FakeTLS прокси для Telegram — одна команда, и готово.

## Установка
```bash
bash <(curl -s https://raw.githubusercontent.com/pink1ep1e/pnk-proxy/main/setup_mtproxy.sh)
```

## Требования
- Ubuntu 20.04 / 22.04 / Debian
- Root доступ
- Открытый порт (по умолчанию 443)
